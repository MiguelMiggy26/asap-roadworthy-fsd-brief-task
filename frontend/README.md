## Getting Started
1. Navigate the backend directory.
- cd backend

2. Install dependencies
- npm install

3. Run the backend on development mode with nodemon
- npm run dev

4. You should see your backend is running at PORT:5000
- http://localhost:5000/



5. On new terminal, navigate the frontend directory.
- cd frontend

6. Install dependencies.
- npm install

7. Run the development server.
- npm run dev

8. You should see your frontend is running at PORT:3000
- http://localhost:3000/
